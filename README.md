## 🙋‍♂️ About Me

I'm a **Data & Analytics Engineer** with **4+ years** of experience designing and building robust data infrastructure — from ingestion pipelines to analytics dashboards. I love turning messy raw data into clean, actionable insights.

- 🔭 I'm currently working on **end-to-end data pipelines** using dbt, Snowflake & AWS/GCP
- 📹 I share knowledge on **YouTube**, **Medium**, and **Instagram** about data engineering & analytics
- 🌐 Check out my portfolio at **[karthikpolisetty.me](https://www.karthikpolisetty.me/)**
- ⚡ Fun fact: I can debug a broken dbt model faster than I can decide what to eat for lunch
- 🎯 I believe in **data quality first** - garbage in, garbage out!
